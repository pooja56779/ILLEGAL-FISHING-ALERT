## ЁЯЫая╕П Troubleshooting

### No voice output?
1. Check volume is not muted
2. Verify "Enable voice alerts" is ON
3. Check browser permissions
4. Try different browser

### Wrong language?
1. Go to Settings
2. Select correct voice language
3. Refresh page

### Voice sounds robotic?
- This is normal for Web Speech API
- For better quality, consider adding Google TTS
